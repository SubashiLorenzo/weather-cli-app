# Weather CLI App

Progetto Node.js che permette di ottenere dati meteo in tempo reale inserendo il nome di una città.

## Funzionalità

- Ricerca città tramite API di geocoding
- Recupero dati meteo tramite Open-Meteo
- Visualizzazione temperatura, vento, umidità e precipitazioni
- Gestione errori per input non valido o città non trovata

## Come eseguire il progetto

## Requisiti

- Node.js 18+

```bash
node weather.js "Roma"
```
